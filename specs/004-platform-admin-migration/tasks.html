Tasks: 플랫폼 관리 기능 이관 (Clear Platform → 에이전트 앱) ​

입력 산출물: plan.md, spec.md, research.md, data-model.md, contracts/rest-api.md, quickstart.md

경로 규약 (타겟 app/):

  • 백엔드: app/src/main/java/com/example/agent/platform/<module>/ (dto/, mapper/, service/, web/)
  • 매퍼 XML: app/src/main/resources/mapper/<module>/
  • 스키마: app/src/main/resources/db/changelog/
  • 테스트: app/src/test/java/com/example/agent/platform/<module>/

테스트 포함: plan §D 가 단위/통합 테스트(H2 Oracle 호환 모드)를 명시 → 스토리별 테스트 태스크 포함.

유저 스토리 ↔ 우선순위 (spec User Scenarios 기준):

스토리우선순위내용기능
