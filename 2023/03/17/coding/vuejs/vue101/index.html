개요

Vue.js는 React.js와 함께 가장 인기있는 자바스크립트 프레임워크 중의 하나이다.
w3techs.com의 2023년 3월 16일 기준 시장점유율을 보면 react 대비 아직 많은
사이트에서 사용되고 있지는 않지만 트래픽이 높은 사이트에서 활용되고 있음을 알수
있다. 1년 넘게 프로젝트를 진행해보면서 느낀건 정말 쉽다는 점이다.

구글 트랜드를 보면 react 가 관심도가 높지만 Vue.js는 React.js 보다 배우기 쉽다는 장점 때문에 지속
성장할거라 예상한다. vue.js를 배워두면 react 스벨트 같은것도 쉽게 배울수 있을리라 예상한다.
