CAT-16. 크리에이티브 & 콘텐츠 ​

평가 핵심: 에이전트가 뉴스레터, 영상 기획, 카피라이팅, 스토리텔링 등 창의적 콘텐츠 제작을 할 수 있는가?

CC-01. 사내 뉴스레터 제작 ⭐⭐ ​

시나리오 배경 ​

매주/매월 발행하는 사내 뉴스레터. 직원들이 실제로 읽게 만드는 것이 핵심.

프롬프트 ​

사내 월간 뉴스레터 4월호를 만들어줘.

회사: IT 기업 "넥스트웨어" (직원 150명)
뉴스레터명: "넥스트레터"
발행: 매월 첫째 주 월요일, 전사 이메일 발송
현재 오픈율: 35% (목표 60%)

4월 소재:
- 회사 뉴스: Q1 매출 목표 초과 달성 (109%), 신규 대형 고객 2사 계약
- 인사: 신규 입사자 5명 소개, 3월 우수사원 (개발팀 이현정)
- 프로젝트: "스마트팩토리" 프로젝트 성공적 1차 오픈
- 행사: 4/20 가족 초청 행사, 4/25 봄 워크숍
- 복지: 5월부터 자기개발비 한도 상향 (50→100만원)
- 가벼운 콘텐츠: 사내 맛집 추천 (회사 근처 점심), 이달의 playlist

뉴스레터 구성:
1. 편집장 인사 (대표 또는 HR) — 3~4줄, 시즌 감성
2. 이번 달 하이라이트 (Q1 실적) — 숫자 강조, 축하 톤
3. Welcome! 신규 입사자 소개 — 이름, 소속, 한 줄 자기소개, 취미
4. 프로젝트 스포트라이트 — 스마트팩토리 프로젝트 스토리
5. 이달의 우수사원 — 선정 이유, 동료 코멘트 (가상)
6. 다가오는 행사 — 캘린더 형식
7. 알림 (자기개발비 상향) — 핵심만 간결하게
8. 가벼운 코너 — 맛집 3곳 + 직원 투표 결과(가상)
9. 에디터 클로징 — 다음 호 예고

톤: 뉴스 보도 아님. 친한 동료가 알려주는 느낌.
형식: HTML 이메일 기준 마크다운으로 작성.

파일: `content/newsletter-april-2024.md`

주의: 
- 오픈율 35%인 이유는 지루하기 때문. 재미 요소 필수.
- 이모지 적절히 사용 OK (과하지 않게)
- "회사가 잘되고 있다"를 강조하되 자화자찬은 금물

평가 포인트 ​

  • [ ] 전체 구성의 리듬감 (무거운↔가벼운 교차)
  • [ ] 편집장 인사의 시즌 감성
  • [ ] 신규 입사자 소개의 개인적 느낌
  • [ ] 프로젝트 스토리텔링 (프로젝트 수행기 형태)
  • [ ] 가벼운 코너의 재미 요소
  • [ ] 오픈율 60% 달성에 기여할 만한 제목(subject line)
  • [ ] 적절한 이모지 사용량

CC-02. 영상 콘텐츠 기획안 ⭐⭐⭐ ​

시나리오 배경 ​

유튜브/인스타 릴스용 영상 기획. 촬영팀에게 전달할 기획서.

프롬프트 ​

회사 유튜브 채널의 영상 시리즈 기획안을 만들어줘.

채널: HR SaaS 기업 "피플웍스" 공식 유튜브
구독자: 1,200명 (6개월 목표: 5,000명)
타깃 시청자: 중소기업 HR 담당자 + 스타트업 대표
목적: 브랜드 인지도 + 리드 생성

시리즈 기획:

1. `content/youtube-series-plan.md` — 시리즈 전체 기획

   시리즈명: "HR 디버깅" (HR 문제를 코드 디버깅하듯 해결)
   콘셉트: 중소기업이 겪는 HR 난제를 실용적으로 풀어주는 시리즈
   에피소드 수: 시즌 1 — 8편
   영상 길이: 8~12분
   촬영: 사내 스튜디오, 진행자 1명 (HR 매니저)
   
   에피소드 목록:
   EP01. "퇴사자가 연쇄로 나올 때 — 이탈 패턴 읽기"
   EP02. "면접에서 좋았는데 입사하면 달라요 — 채용 미스매치 줄이기"
   EP03. "연봉 협상 요청이 왔을 때 — 보상 정책 설계 가이드"
   EP04. "팀장이 되긴 했는데 뭘 해야 하지 — 신임 매니저 서바이벌"
   EP05. "인사평가 시즌이 또 왔다 — 덜 고통스러운 평가법"
   EP06. "재택이요? 출근이요? — 하이브리드 근무 정책 만들기"
   EP07. "직원이 10명 → 50명 될 때 — 조직 구조 설계"
   EP08. "HR 담당자가 번아웃일 때 — HR의 HR"

2. EP01 상세 기획서

   A. 기본 정보
   - 제목, 길이, 태그, 썸네일 콘셉트
   
   B. 구성 (시간별)
   - 0:00~0:30 — 후킹 (충격적 통계 or 상황 재연)
   - 0:30~2:00 — 문제 정의 ("연쇄 퇴사가 뭔지")
   - 2:00~5:00 — 원인 분석 (3가지 패턴)
   - 5:00~8:00 — 해결 방법 (실전 가이드)
   - 8:00~9:00 — 체크리스트 정리
   - 9:00~9:30 — CTA ("구독/좋아요 + 다음 편 예고")
   
   C. 스크립트 (전체 대사)
   - 진행자 대사 전문
   - 화면에 보여줄 자막/인포그래픽 지시
   - B-roll 촬영 지시 (사무실 풍경, 빈 자리 등)
   
   D. 썸네일 & 제목 A/B 테스트
   - 제목 3안 + 썸네일 콘셉트 3안
   - 클릭률 높은 요소 분석
   
   E. SEO
   - 타깃 키워드 5개
   - 영상 설명문 (SEO 최적화)
   - 태그 목록

파일: `content/youtube-series-plan.md`

CC-03. 카피라이팅 실전 ⭐⭐ ​

시나리오 배경 ​

다양한 맥락의 카피를 짧은 시간에 대량으로 생성하는 능력.

프롬프트 ​

아래 상황별로 카피를 작성해줘. 총 30개.

브랜드: "릴랙스" — 수면 보조 앱 (명상, 수면 음악, 수면 추적)
타깃: 25~45세, 수면에 문제를 느끼는 직장인
톤: 차분하고 따뜻한, 압박 없는

카피 유형:

A. 앱스토어 설명 (3개)
- 앱 이름 옆에 나오는 한 줄 설명 (15자 이내)
- 상세 설명 첫 3줄 (다운로드 결정적 영향)
- "새로운 기능" 업데이트 노트 (수면 리포트 기능 추가)

B. 푸시 알림 (5개)
- 저녁 9시 수면 루틴 리마인더
- 7일 연속 사용 축하
- 30일 미접속 재활성화
- 새 명상 콘텐츠 추가 알림
- 수면 분석 리포트 준비 완료

C. 인앱 팝업 (4개)
- 무료→유료 전환 유도 (체험 종료 3일 전)
- 리뷰 요청 (5일 연속 사용 후)
- 친구 추천 이벤트
- 연간 구독 할인 프로모션 (40% off)

D. 인스타그램 캡션 (5개)
- 월요일: 한 주 시작 응원
- 수요일: 수면 팁 (블루라이트 관련)
- 금요일: 주말 릴랙스 루틴 제안
- 시즌: 벚꽃 시즌 수면 명상
- 유저 후기 리그램 캡션

E. 이메일 제목줄 (8개)
- 가입 환영
- 사용 가이드
- 비활성 사용자 재참여
- 유료 전환 유도
- 할인 프로모션
- 수면 분석 리포트 발송
- 새해 특별 캠페인
- 해지 방지 (마지막 설득)

F. 광고 카피 (5개)
- Google 검색 광고 (제목 30자 + 설명 90자)
- 인스타 피드 광고 (본문 + CTA)
- 유튜브 범퍼 광고 (6초 대사)
- 카카오 비즈보드 (한 줄 카피)
- 네이버 배너 (헤드라인 + 서브)

모든 카피에:
- 왜 이렇게 썼는지 한 줄 의도 설명
- 대안 카피 1개씩 (A/B 테스트용)

파일: `content/relax-copywriting-collection.md`

평가 포인트 ​

  • [ ] 30개 카피의 톤 일관성 (차분하고 따뜻한)
  • [ ] 채널별 길이/형식 준수 (푸시 알림 ≠ 이메일 ≠ 광고)
  • [ ] 전환 유도 카피의 설득력 (압박 없이)
  • [ ] 재활성화 카피의 공감 (죄책감 유발 안 함)
  • [ ] 유튜브 6초 범퍼 대사의 임팩트
  • [ ] A/B 대안의 차별화
  • [ ] 의도 설명의 마케팅 논리

CC-04. 기업 블로그 콘텐츠 전략 ⭐⭐ ​

시나리오 배경 ​

SEO와 리드 생성을 고려한 블로그 콘텐츠 전략.

프롬프트 ​

B2B SaaS 기업의 블로그 콘텐츠 전략을 수립하고 샘플 포스트를 작성해줘.

회사: "체크메이트" — 법인카드 경비 관리 SaaS
타깃: 중소기업 재무/총무 담당자, CFO
현재: 블로그 없음, SEO 유입 0
목표: 6개월 내 월 5,000 세션, 리드 50건/월

산출물:

1. `content/blog-content-strategy.md` — 콘텐츠 전략
   
   A. 키워드 리서치 (가상 데이터)
   - 타깃 키워드 30개 (검색량, 경쟁 강도, 의도)
   - 키워드 클러스터 5개 (주제별 그룹)
   - 우선순위 (검색량 × 전환 가능성 × 경쟁 강도)
   
   B. 콘텐츠 퍼널
   - TOFU (인지): "법인카드 관리 방법", "경비 정산 팁"
   - MOFU (고려): "경비 관리 SaaS 비교", "엑셀 vs 전용 툴"
   - BOFU (결정): 고객 사례, 제품 비교, 무료 체험 안내
   
   C. 편집 캘린더 (6개월)
   - 월 4개 포스트 (총 24개)
   - 주제, 키워드, 퍼널 단계, 예상 발행일
   
   D. 콘텐츠 가이드라인
   - 톤, 구조, 길이 (2,000~3,000자)
   - SEO 체크리스트 (제목, 메타, H2, 내부링크)
   - CTA 삽입 규칙

2. `content/blog-sample-post.md` — 샘플 포스트 1편
   
   제목: "법인카드 정산, 아직도 엑셀로 하세요? — 5가지 대안 비교"
   - 키워드: "법인카드 정산"
   - 퍼널: MOFU
   - 길이: 약 2,500자
   - 구성: 도입(공감) → 현재 방식의 문제 → 5가지 대안 비교표 → 
     선택 가이드 → CTA (무료 체험)
   - SEO 요소 모두 포함 (H1, H2, 메타 설명, ALT 텍스트 위치)
   - 마지막에 SEO 체크리스트 점검 결과

CC-05. 위기 커뮤니케이션 시나리오 ⭐⭐⭐ ​

시나리오 배경 ​

PR 위기 상황에서의 대외 커뮤니케이션. 한 번의 실수가 브랜드를 망칠 수 있는 상황.

프롬프트 ​

아래 위기 상황에 대한 커뮤니케이션 계획을 만들어줘.

상황:
- 우리: 건강식품 이커머스 "내추럴라이프"
- 사건: 판매한 유기농 주스에서 이물질(금속 파편) 발견
- 경위: 고객 1명이 음용 중 발견 → SNS에 사진 올림 → 2시간 만에 조회 5만
- 현재 상태: 해당 상품 200병 판매됨, 다른 피해 신고 아직 없음
- 원인: 아직 조사 중 (제조사 OEM, 우리는 유통)
- 시간: 지금 오후 3시, SNS 반응이 실시간으로 커지고 있음

산출물:

1. `content/crisis-communication-plan.md`

   A. 즉시 대응 (오후 3시~6시)
   - 내부 비상 회의 소집 (누가, 어디서)
   - 팩트 확인 체크리스트 (지금 확인할 수 있는 것)
   - SNS 1차 공식 대응 (30분 내 게시)
     - 게시할 내용 (인지+조사중+고객 안전 최우선)
     - 게시하면 안 되는 내용 (원인 추측, 책임 회피)
   - 고객 연락 (피해 고객 직접 연락, 통화 스크립트)
   - 해당 상품 즉시 판매 중지

   B. D+1 대응
   - 공식 사과문 (홈페이지 + 이메일 + SNS)
   - 전량 리콜 공지 (환불 절차)
   - 언론 대응 (보도자료 or 기자 문의 Q&A)
   - 식약처 신고 (자진 신고)

   C. D+3~7 대응
   - 원인 조사 결과 공유 (투명하게)
   - 재발 방지 대책 발표
   - 피해 고객 보상안 (환불 + α)
   - 신뢰 회복 캠페인 (품질 검사 과정 공개)

2. `content/crisis-templates.md` — 템플릿 모음
   - SNS 1차 대응문 (인스타/트위터/페이스북 각각)
   - 공식 사과문 (홈페이지 게시용)
   - 고객 이메일 (구매자 전체)
   - 보도자료
   - 내부 직원 공지 (직원이 외부에서 질문받을 때 대응 가이드)
   - 고객 전화 문의 응대 스크립트

3. `content/crisis-do-dont.md` — 위기 대응 절대 원칙
   - DO: 빠른 인정, 투명한 공유, 고객 안전 최우선
   - DON'T: 삭제/은폐, 책임 전가("제조사 잘못"), 침묵, 법적 대응 위협
   - 소셜 미디어에서 절대 하면 안 되는 행동 5가지
   - 과거 기업 위기 대응 사례 (잘한 것 / 못한 것)
